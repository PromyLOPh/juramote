juramote
========

Remote control for Jura coffee maker Xs90 (EF516M V01.25).

Physical connection
-------------------

Connector on the back of the machine is female 9 pin d-sub.

5       1
 ───────
 \     /
  ─────
 9     6

1: TX (output)
2: GND
3: RX (input)
4: VCC (5V)
